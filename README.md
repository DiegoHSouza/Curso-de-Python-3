# 🐍 Curso de Python 3 do Zero ao Avançado + Projetos Reais

Repositório oficial dos meus estudos no curso **"Python 3+ Completo"** da Udemy. Este repositório documenta minha evolução técnica, desde a lógica básica até o deploy de aplicações web complexas e Design Patterns.

## 🚀 Projetos Principais (Destaques)
Aqui estão as aplicações completas desenvolvidas durante o curso:

| Projeto | Tecnologias | Descrição |
| :--- | :--- | :--- |
| **Agenda** | Django, SQLite | Sistema de gestão de contatos com autenticação de usuário e CRUD completo. |
| **Blog** | Django, Summernote | Plataforma de blog com painel administrativo, comentários e categorias. |
| **Loja Virtual** | Django, MySQL | E-commerce completo com carrinho de compras, gestão de pedidos e pagamentos. |
| **Calculadora** | PySide6 (Qt) | Aplicação Desktop com interface gráfica nativa simulando o Windows Calculator. |
